# Student Chatbot v2 🎓🤖

نسخه دوم چت‌بات دانشجویی با ساختار بهبود‌یافته و قابلیت‌های توسعه‌پذیرتر.

## ✨ تفاوت‌های نسخه v2
- بهبود ساختار فرانت‌اند و بک‌اند
- آماده برای توسعه چند سرویس
- مدیریت بهتر کانفیگ‌ها و متغیرهای محیطی
- مناسب برای دیپلوی روی Render

## 🛠 تکنولوژی‌ها
- Backend: Python (FastAPI)
- Frontend: (React / HTML / ...)
- API: REST
- Deployment: Render
